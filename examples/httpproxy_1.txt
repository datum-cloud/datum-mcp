Create an HTTPProxy named bittensor‑dapp that terminates TLS for cloud.datum.net,
rate‑limits clients to 100 rps (burst 20), allows CORS from Datum origins,
blocks Hawaii CIDRs, strips /api and forwards to https://api.cloud.datum.net,
and serves the UI from https://static.cloud.datum.net.
---
# NOTE: The following features are NOT in the Datum HTTPProxy schema and are
# therefore omitted here: TLS termination, rate limiting, CORS, CIDR blocking,
# active health‑checks.
apiVersion: networking.datumapis.com/v1alpha
kind: HTTPProxy
metadata:
  name: bittensor-dapp
spec:
  rules:
    - name: api-route
      matches:
        - path:
            type: PathPrefix
            value: /api
      backends:
        - endpoint: https://api.cloud.datum.net
          filters:
            - type: URLRewrite
              urlRewrite:
                path:
                  type: ReplacePrefixMatch
                  replacePrefixMatch: /
    - name: ui-route
      matches:
        - path:
            type: PathPrefix
            value: /
      backends:
        - endpoint: https://static.cloud.datum.net
